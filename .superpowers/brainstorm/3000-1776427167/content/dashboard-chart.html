实现方式

不用第三方图表库，用纯 CSS bar chart（和现有 Heatmap 风格一致），每 5 秒刷新时重绘。

  • 数据粒度：每 5 分钟一个柱（从 events.jsonl 按时间戳聚合）
  • 窗口：显示最近 2 小时（24 根柱子）
  • 颜色：按事件类型分层 — turn_end 用 #60a5fa（蓝），agent 事件用 #fbbf24（金）
  • hover 提示：显示该时段的事件数和 session ID
  • 点击：跳转到新的统一 Sessions 页面
